Go Mmp

Mini Macro Pad go-mmp Macro Pad driver software, written in Golang. If you have an arduino powered device, you could use this to run various keyboard shortcuts and other macros. No device? No problem! You can still click on the buttons to run the macros. Here’s what the GUI looks like, you can click the buttons to run the macro, or use the arduino to press them.
